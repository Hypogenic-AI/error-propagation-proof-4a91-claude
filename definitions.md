# Definitions and Notation

## Proof State Space

**Definition 1 (Proof State Space).** A *proof state space* is a pair (S, d) where S is a set of proof states and d: S × S → [0,∞) is a metric measuring the "distance from correctness." Each s ∈ S encodes the current set of derived propositions, assumptions, and logical context. We denote by s* ∈ S the unique correct (target) proof state, satisfying d(s*, s*) = 0.

**Definition 2 (Reasoning Operation).** A *reasoning operation* is a map T: S → S representing a single proof step (e.g., modus ponens, substitution, case analysis). The set of all reasoning operations is denoted 𝒯.

**Definition 3 (Error Operator).** Given a correct reasoning operation T and an actual (possibly erroneous) operation T̃, the *error operator* is E = T̃ ∘ T⁻¹: S → S (when T is invertible). The error operator measures the deviation introduced by one step.

## Group Structure

**Definition 4 (Error Group).** Let G = (𝒢, ∘) be the group generated by all error operators under composition:
- Closure: E₁ ∘ E₂ ∈ 𝒢 for all E₁, E₂ ∈ 𝒢
- Associativity: (E₁ ∘ E₂) ∘ E₃ = E₁ ∘ (E₂ ∘ E₃)
- Identity: Id ∈ 𝒢 (the identity represents no error)
- Inverses: For each E ∈ 𝒢, ∃E⁻¹ ∈ 𝒢 with E ∘ E⁻¹ = Id (error correction)

**Definition 5 (Error Propagation Sequence).** An *n-step error propagation sequence* is the composition Eₙ ∘ Eₙ₋₁ ∘ ··· ∘ E₁ where each Eᵢ ∈ G is the error at step i. The cumulative error after n steps is E⁽ⁿ⁾ = ∏ᵢ₌₁ⁿ Eᵢ (product in G).

**Definition 6 (Abelian Error Propagation).** An error propagation sequence has *abelian error propagation* if all error operators pairwise commute: EᵢEⱼ = EⱼEᵢ for all i,j. Equivalently, the subgroup ⟨E₁,...,Eₙ⟩ ≤ G is abelian.

## Linear Representation and Spectral Analysis

**Definition 7 (Error Propagation Operator).** Given a probability distribution μ on the error group G, the *error propagation operator* is the linear map P_μ: ℂ[G] → ℂ[G] on the group algebra defined by:
  P_μ(f)(g) = Σ_{h∈G} μ(h) f(h⁻¹g)
This is the convolution operator (left regular representation).

**Definition 8 (Spectral Radius of Error Propagation).** The *spectral radius* of error propagation is ρ(P_μ) = max{|λ| : λ is an eigenvalue of P_μ acting on ℂ[G]/ℂ·1_G} (excluding the trivial eigenvalue 1 from the constant functions).

**Definition 9 (Convergence Rate).** The *convergence rate* of an error propagation process is r = -log ρ(P_μ). Larger r means faster convergence to the stationary (correct) distribution. The mixing time satisfies t_mix = O(1/r · log(1/ε)).

## Commutator Structure

**Definition 10 (Commutator Norm).** For a finite group G with error propagation operator P_μ, the *commutator norm* is:
  κ(G, μ) = max_{ρ non-trivial irrep} (dim ρ - 1) · ‖ρ̂(μ)‖
where ρ̂(μ) = Σ_g μ(g)ρ(g) is the Fourier transform of μ at representation ρ. For abelian G, dim ρ = 1 for all ρ, so κ = 0.

**Definition 11 (Derived Series Length).** The *derived series* of G is G = G⁽⁰⁾ ⊇ G⁽¹⁾ ⊇ G⁽²⁾ ⊇ ··· where G⁽ⁱ⁺¹⁾ = [G⁽ⁱ⁾, G⁽ⁱ⁾]. The *derived length* d(G) is the smallest k with G⁽ᵏ⁾ = {e}. Abelian groups have d(G) = 1. Non-abelian solvable groups have d(G) ≥ 2.
